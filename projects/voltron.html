VOLTRON Common Data Model


The VOLTTRON Transactive Energy Platform provides a general purpose message bus but does not define message content or processing semantics. While agents and services may be able to send messages across the bus, they cannot communicate without implementing a common protocol and data model. This project will create a protocol adapter layer (potentially implemented as Proxy Agents) on top of the VOLTTRON Message Bus that will enable the reuse of existing, widely used protocols between VOLTTRON agents and services. The message bus will provide a common transport for the message content of protocols such as SEP2, OpenADR, and OCPP, and information models based on IEC61850, MESA/SunSpec Inverter, and Storage and Meter Data Models.
